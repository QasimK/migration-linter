# Migration Linter

Check whether migrations will cause an outage.

## Usage

* M1xx = Global errors
* M2xx = Statement errors
* M3xx = Warnings

## Development

1. `poetry install` to set up the virtualenv (one-off)
2. `make fix`, `make check`, and `make test`, before committing
